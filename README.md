## Key Concepts Cheat Sheet

### EC2 Pricing Models
- **On-Demand**: No commitment, pay by hour
- **Reserved**: 1-3 year commitment, up to 72% savings
- **Spot**: Up to 90% savings, can be interrupted
- **Dedicated**: Physical server for compliance needs

### Shared Responsibility Model
- **AWS**: Physical security, hardware, networking
- **You**: Data, IAM, OS patching, firewall rules
